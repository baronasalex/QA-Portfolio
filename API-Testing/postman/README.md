## Project Overview
API testing collection created using Postman to validate CRUD operations on a public REST API.

## Tested Functionality
- GET user
- POST create user
- PUT update user
- DELETE user

## Testing Approach
- Status code validation
- Response body validation
- Dynamic data handling using collection variables
- Automated execution via Collection Runner

## Tools
- Postman
- REST API
- JavaScript assertions

## How to Run
1. Import collection into Postman
2. Add variables:
   - baseUrl = https://reqres.in
   - apiKey = your_api_key
3. Run collection using Runner
